# 🍱 HomeSavour

> Empowering Home Chefs. Delivering Homemade Happiness.

HomeSavour is a full-stack mobile marketplace that connects talented home chefs (primarily mothers) with customers looking for fresh, healthy, homemade food. Customers can browse nearby home kitchens, place orders, and pick up meals directly from the chef's location.

---

## 📱 Features

### 👩‍🍳 Home Chef Features
- Secure registration and login
- Create and manage food listings
- Upload dish images
- Set price, quantity, and availability
- Accept or reject customer orders
- Update order status
- View daily earnings dashboard
- Manage profile and kitchen details

### 🧑‍🍽️ Customer Features
- Browse nearby home chefs
- Search dishes by category
- View dish details and chef information
- Add items to cart
- Place pickup orders
- Track order status in real-time
- Review and rate chefs
- Manage profile and order history

### 📦 Order Management
- Cart functionality
- Pickup-based ordering system
- Order status tracking
- Order history for customers and chefs

### 🔔 Notifications
- Order placed
- Order accepted
- Food ready for pickup
- Order completed

### 📍 Location Services
- GPS-based nearby chef discovery
- Pickup address management
- Location filtering

### 💳 Payments
- UPI Payments
- Razorpay Integration (Optional)
- Cash on Pickup

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Johnkishore11/HomeSavour.git
cd HomeSavour
```

### Frontend Setup

```bash
cd frontend

npm install

npx expo start
```

### Backend Setup

```bash
cd backend

npm install

npm run dev
```

## 🎯 Problem Statement

Many talented home chefs struggle to reach customers despite preparing high-quality homemade meals. At the same time, customers seek healthier alternatives to restaurant food.

HomeSavour bridges this gap by providing a trusted platform where home chefs can earn income while customers enjoy authentic homemade food.
